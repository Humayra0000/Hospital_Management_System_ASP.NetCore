# Hospital Management System (HMS)

Project Description

This project primarily focuses on backend development and server-side logic using ASP.NET Core and MVC architecture. The frontend/UI design was not a major focus of this project; instead, the emphasis was on implementing business logic, database design, and system functionality.
The system demonstrates how real-world hospital data can be managed efficiently using a centralized backend application.

Features

- Patient Management
Create, update, and manage patient records in the database.

- Doctor Management
Store and maintain doctor information and availability.

- Appointment Management
Handle appointment scheduling between patients and doctors.

- Role-Based Access Control
Backend support for Admin, Doctor, and Staff roles.

- Database Operations
CRUD operations using SQL Server with Entity Framework Core.

Technologies Used

- Backend: ASP.NET Core
- Framework: ASP.NET Core MVC
- Database: SQL Server
- ORM: Entity Framework Core
- Development Tool: Visual Studio

Project Purpose

This project was developed for academic and learning purposes, with a strong emphasis on:

- ASP.NET Core backend development
- MVC architecture
- Database integration and CRUD operations
- Understanding real-world hospital management workflows
- Future Enhancements
- Improved frontend UI/UX
- REST API integration
- Authentication using JWT
- Reporting and analytics modules
